CASE STUDY 01

Spendly

Project Type FinTech
Role Product Designer & Frontend Developer
Core Features Expense Tracking & Savings Analysis
Built With Figma • Antigravity • Firebase

The Challenge

Managing monthly expenses is difficult for many college students and freshers due to irregular spending habits, limited financial awareness, and the lack of simple budgeting tools designed specifically for young users. Most existing finance apps feel overly complex, cluttered, or disconnected from student lifestyles.

Spendly was designed to solve this problem through a clean, intuitive, and student-friendly platform that simplifies expense tracking, recurring payment management, savings monitoring, and monthly spending analysis.

Architectural Highlights

The platform was designed with a modern FinTech-inspired interface focused on clarity, accessibility, and smooth user interaction. Responsive layouts, minimal UI components, and interactive expense visualizations were implemented to create a seamless budgeting experience across devices.

The project combined thoughtful UI/UX planning in Figma with frontend development to deliver a visually engaging and user-centric financial management platform.

Technology Stack

HTML5 CSS3 JavaScript Figma Antigravity Firebase Node.js Git & GitHub

Project Deliverables

  • Expense Tracking Dashboard
  • Savings & Spending Analytics
  • Recurring Payment Reminders
  • Responsive UI/UX Design
  • Interactive Financial Insights
  • Student-Centric Budget Management System
